        # Travel Wisata Lokal - Flutter
        Proyek contoh aplikasi travel wisata lokal untuk tugas besar Mobile Programming.

        ## Fitur
        - CRUD destinasi (lokal) menggunakan SQLite (sqflite)
        - Input gambar dari galeri (image_picker)
        - Menampilkan marker destinasi di halaman peta (google_maps_flutter) â€” perlu API key untuk menampilkan peta.
        - Form input jam buka/tutup dan koordinat latitude/longitude.

        ## Cara menjalankan
        1. Pastikan Flutter SDK terpasang.
2. Jalankan `flutter pub get`.
3. Jalankan `flutter run`.

Catatan: tambahkan Google Maps API key di AndroidManifest (Android) dan Info.plist (iOS) jika ingin fitur peta aktif.
