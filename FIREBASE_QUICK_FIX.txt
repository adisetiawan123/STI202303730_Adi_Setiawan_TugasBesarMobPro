# Quick Reference: Firebase Setup Required

## Problem
App shows: `API key not valid. Please pass a valid API key.`

## Causes
1. `firebase_options.dart` uses dummy key: `AIzaSyDummyKeyForDevelopment`
2. Missing: `android/app/google-services.json`

## Solution (5 minutes)

### 1. Get Real Firebase Key
- Go to [Firebase Console](https://console.firebase.google.com)
- Project Settings → API Keys → Copy "Browser key" (starts with `AIzaSy...`)

### 2. Update firebase_options.dart
Replace `AIzaSyDummyKeyForDevelopment` with your real key in 4 places:
- `web` constant
- `android` constant
- `ios` constant
- `macos` constant

### 3. Get google-services.json
- Firebase Console → Project Settings → Download `google-services.json`
- Place at: `android/app/google-services.json`

### 4. Rebuild
```powershell
flutter clean && flutter pub get && flutter run
```
